Machine Learning Enhancement

ML 도입으로 달라진 VE 자문 엔진

키워드 사전 검색기에서 의미 이해 기반 지능형 자문 엔진으로 전환. 4개 Tier ML 파이프라인이 검색 정확도와 분류 자동화를 혁신합니다.

TIER 1 / SEMANTIC

Embedding 시맨틱 검색

384차원 벡터 공간에서 코사인 유사도 기반 의미 검색. "지붕 물 새는 문제" → "옥상 방수 공법" 자동 매칭. 키워드 외 질의 처리율 0% → 100%.

TIER 2 / HYBRID RAG

복합 스코어 + Gemini RAG

Embedding(60%) + KG 구조(40%) 복합 재정렬. Gemini 2.0 Flash가 환각 방지 프롬프트로 데이터 기반 분석 답변을 생성합니다. 응답 시간 0.07초.

TIER 3 / CLASSIFIER

자동 분류 ML

클러스터링 인사이트
