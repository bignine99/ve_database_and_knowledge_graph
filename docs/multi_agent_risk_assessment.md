## 2. 핵심 리스크 5가지

### ① LLM 환각 (Hallucination)

| 항목 | 내용 |
|---|---|
| **위험** | AI가 존재하지 않는 실적/수치를 생성 |
| **예시** | "태양광 연간 절감 5,000만원" → 실제 3,200만원 |
| **현재 대응** | Gemini temperature 0.3, 시스템 제약 4조, 데이터 기반만 인용 강제 |
| **추가 필요** | 비용 수치는 내역 AI 데이터만 사용, AI 추정 금지 |

### ② 727건 데이터 편향 (Bias)

| 항목 | 내용 |
|---|---|
| **위험** | 대형 프로젝트 편중, 조경 5% vs 건축 50%, 2015~2020 데이터 60% |
| **예시** | 소규모 지방 주택 → "유사 사례 없음" |
| **현재 대응** | HOW1 대분류(58%)로 최소한의 분류, 클러스터 통계 제공 |
| **추가 필요** | 데이터 부족 시 "표본 N건, 참고만 사용" 경고 자동 표시 |

### ③ 현장 여건 미반영

| 항목 | 내용 |
|---|---|
| **위험** | 도로 폭, 기술자 숙련도, 자재 수급 등 로컬 요인 무시 |
| **예시** | "프리캐스트 공법" 제안 → 현장 도로 좁아서 반입 불가 |
| **현재 대응** | Domain Agent가 SKILL 기반 기술 검증 수행 |
| **추가 필요** | `site_conditions` 입력 스키마 추가 (Phase 2) |

### ④ "이미 아는 내용" 문제

| 항목 | 내용 |
|---|---|
| **위험** | 경력 3년차도 아는 뻔한 아이디어만 도출 |
| **예시** | "천연석 → 인조석", "LED 교체" |
| **현재 대응** | DB 유사 사례 기반으로 과거 성공 사례의 구체적 수치 제공 |
| **차별점** | 아이디어 자체보다 **"근거 + 수치 + 보고서 초안"** 자동 생성이 핵심 가치 |

### ⑤ API 비용

| 항목 | 내용 |
|---|---|
| **위험** | Gemini API 호출 과다 → 비용 급증 |
| **현재 수준** | 1세션 ~8회 API 호출 (Idea 1 + Domain 6 + Report 1) |
| **대응** | Gemini Flash 사용 (저비용), 도메인 검증은 상위 5개만, 캐싱 검토 |

## 4. 이 시스템이 도움이 되는 곳 / 안 되는 곳

### ✅ 적극 추천

| 용도 | 이유 |
|---|---|
| 초기 아이디어 도출 단계 | 2~3주 → 20분, 시간 절감 최대 |
| 표준 프로젝트 (아파트, 학교) | 유사 사례 풍부 |
| 초급 VE 인력 지원 | Senior 수준의 체크리스트 자동 제공 |
| 보고서 초안 작성 | 양식 + 근거 + 수치 자동 삽입 |

### ⚠ 신중히 고려

| 용도 | 이유 |
|---|---|
| 대형 복잡 프로젝트 | 전문가 검증 비중 ↑, AI 기여도 ↓ |
| 특수 현장 조건 | 현장 여건 입력 미지원 |

### ❌ 부적절

| 용도 | 이유 |
|---|---|
| AI 결과만으로 의사결정 | 환각 + 현장성 부재 |
| 법적 책임 있는 분석 | AI에 책임 귀속 불가 |
| 정밀 비용 산출 | 내역 AI 없이는 추정 수준 |

## 5. 실무 적용 원칙 (반드시 준수)

```
원칙 1: AI는 초안, 사람이 확정
  → AI 분석 → Domain Expert 검증 → 시공사 현장 협의 → 확정
  → (금지) AI 분석 → 즉시 실행

원칙 2: 모든 출력에 "AI 초안" 표시
  → "⚠ AI 초안 — 반드시 전문 엔지니어의 검토를 거친 후 사용하세요"

원칙 3: 근거 없으면 말하지 않기
  → DB에 없는 수치 생성 금지
  → "데이터에서 확인되지 않음" 명시

원칙 4: 신뢰도를 항상 표시
  → confidence < 0.35 → "불확실 — 후보 참고"
  → 표본 < 10건 → "표본 부족 — 참고만 사용"
```

## 8. 결론

> **"지금 당장 완벽한 솔루션은 아니지만, 올바르게 개발하면 VE 산업을 변화시킬 잠재력이 있다."**

핵심 가치는 **"VE 품질 향상"보다 "VE 속도 향상 + 표준화"**입니다.

- 현재: 주니어 VE 컨설턴트 수준의 초안 자동 생성
- 6개월 후: 유사 사례 기반 근거 품질 대폭 향상
- 2년 후: 현장 맞춤형 분석 + 자동 협의 시스템

**추천 전략: "파일럿 → 점진적 확대"** — 첫 6개월은 조율 기간으로 70% 수준에서 시작하고, 실제 VE 결과 피드백을 축적하여 지속 개선합니다.
